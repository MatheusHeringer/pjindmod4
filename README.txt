Existem outras entidades além dessas três?
-Sim, existe a entidade professores e a possibilidade de se ter mais algumas.

Quais são os principais campos e tipos?
-Os principais campos são as chaves primárias de cada entidade. Chaves primarias são campos identificadores,os principais identificadores de cada entidade são:
Curso -> código do curso - tipo varchar
Alunos -> matricula - tipo varchar
turma-> código da turma - tipo varchar
professores -> matricula - tipo varchar

Como essas entidades estão relacionadas?
- Através de relacionamentos que variam de muitos pra muitos ou um pra muitos.
